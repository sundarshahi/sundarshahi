I take things from an empty repo to something running in production — the interface, the
backend, and the infrastructure underneath it. Six years of that so far, mostly SaaS.

Lately I've been spending my time on developer tools and LLM plumbing:

- **[llm-gateway](https://github.com/sundarshahi/llm-gateway)** — run Claude and Gemini behind your own server. Go library + HTTP gateway, with OAuth, caching, and clean JSON out.
- **[drydock](https://github.com/sundarshahi/drydock)** — an agent pipeline that walks a product from idea to launch inside Claude Code.
- A few small tools I built because I was tired of not having them: [image-kit](https://github.com/sundarshahi/image-kit), [unboilr](https://github.com/sundarshahi/unboilr), [i18n-sheet](https://github.com/sundarshahi/i18n-sheet).

The day job has looked like cloud telephony (KrispCall), AI storytelling (SparkTales), and a
coastal-shipping platform (Keelspot). I architect on GCP and AWS, and I'm a Google Cloud
certified Professional Cloud Architect. Mostly I just like the part where a messy idea turns
into something people actually use.
